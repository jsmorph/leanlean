This project aims to draft a minimal Lean 4 specification and then to
implement a Lean kernel in Lean 4 that follows that specification.
The work is spec-driven.  It does not derive the kernel from the Lean 4
sources or treat the existing implementation as the specification.

1. Draft an initial minimal Lean 4 specification through research.
   The first version must include inductive types.

1. Implement a minimal kernel in standard Lean 4 for that
   specification.  The result should constitute a serious proof of
   concept.
